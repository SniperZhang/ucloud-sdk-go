# UCloud Golang SDK

UCloud Golang SDK is a toolkit for developer to create custom infrastructure via ucloud open api.

## Install

### via source

```go
go get github.com/ucloud/ucloud-sdk-go
```

### via dep

```
dep ensure -add github.com/ucloud/ucloud-sdk-go
```
